# Frontend

## Cài đặt & Chạy

```bash
# 1. Cài đặt thư viện
npm install

# 2. Chạy ứng dụng
npm run dev
```

Truy cập tại: `http://localhost:3000`

## Cấu hình (.env.local)

Tạo file `.env.local`:

```env
NEXT_PUBLIC_API_URL=http://localhost:4000/api
NEXT_PUBLIC_SOCKET_URL=http://localhost:4000
NEXT_PUBLIC_APP_URL=http://localhost:3000
```
