# Admin Dashboard

## Cài đặt & Chạy

```bash
# 1. Cài đặt thư viện
npm install

# 2. Chạy ứng dụng
npm run dev
```

Truy cập tại: `http://localhost:3001`

## Cấu hình (.env.local)

Tạo file `.env.local`:

```env
NEXT_PUBLIC_API_BASE_URL=http://localhost:4000
NEXT_PUBLIC_API_BASE=http://localhost:4000
```
